# 🎓 UG Life Tracker

A minimal, aesthetic web app to **track and reflect on your undergraduate journey week by week**.

Instead of letting 4 years pass unnoticed, this tool helps you **measure your consistency, productivity, and growth** over time.

---

## ✨ Features

- 📅 Generate a full **4-year UG calendar (8 semesters)**
- 🧠 **Weekly self-scoring system** (1–5 scale)
- 📊 Real-time stats:
  - Weeks elapsed
  - Weeks scored
  - Average score
  - Weeks remaining
- 🎯 Highlight:
  - Current week
  - Past weeks
  - Future weeks
- 🏖️ Separate **semester & break periods**
- ⚙️ Customize semester dates manually
- 💾 Data stored locally using **localStorage**
- 🎨 Clean, minimal UI (Notion/GitHub style)

## 🧠 How It Works

1. Enter your **UG start date**  
2. App automatically:
   - Splits 4 years into semesters & breaks  
   - Generates weekly blocks  
3. Click any **past week** → assign a score  
4. View your **progress stats instantly**
